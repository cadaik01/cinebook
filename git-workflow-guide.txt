1. BASIC WORKFLOW (SOLO DEVELOPER)
===============================================

üîÑ DAILY WORKFLOW:
----------------------
1. Open terminal in project directory
2. Edit/create code in VS Code
3. Run these 3 commands:

   git add .
   git commit -m "Brief description of changes"
   git push

EXAMPLE:
------

git commit -m "Add Movie model and migration"
git push

--------------------

üö® COMPLETE WORKFLOW:
------------------

4. git commit -m "..."   # Commit with clear message
5. git push              # Push code for team

===============================================
3. HANDLING CONFLICTS
===============================================

‚ùó WHEN CONFLICTS OCCUR:
----------------------
1. git pull              # Will report conflicts
2. Open conflicted files in VS Code
3. Find and fix these lines:
   <<<<<<< HEAD
   [your code]
   =======
   [other person's code]
   >>>>>>> commit-id

4. Edit to correct code (remove conflict markers)
5. git add .
6. git commit -m "Fix conflict in [filename]"
7. git push

4. BRANCHING STRATEGY (FOR LARGE TEAMS)
===============================================

üåø CREATE BRANCH FOR NEW FEATURE:

üìù WORKING IN BRANCH:

üîÑ MERGE TO MAIN:

===============================================
7. USEFUL GIT COMMANDS
===============================================

üìä CHECK STATUS:
----------------------
git status               # View changed files
git log --oneline        # View commit history
git branch               # View branches
git remote -v            # View remote repository

üîÑ SYNC AND UPDATE:
----------------------
git pull                 # Get latest code from remote
git fetch                # Get new info (without merge)
git merge origin/main    # Merge code from remote

üåø BRANCH MANAGEMENT:
-----------------
git checkout -b new-branch    # Create and switch to branch
git checkout main             # Switch to main branch
git branch -d branch-name     # Delete local branch
git push origin --delete branch-name  # Delete remote branch

‚è™ UNDO CHANGES:
-------------------
git checkout -- file.txt     # Undo uncommitted file
git reset HEAD~1              # Undo last commit
git revert commit-id          # Create revert commit
